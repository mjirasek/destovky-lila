Destovky Lila

This repository contains a trimmed copy of the Lichess/Lila codebase for experimenting with Lila-style accounts, lobby, challenges, and game infrastructure for Destovky.

Heavy optional assets such as NNUE files, speech models, and background gallery files are intentionally omitted. The real application is still a Scala/Play server and is not hosted by GitHub Pages.

Source: github.com/mjirasek/destovky-lila
